# FeelWrite Habit Tracker

Personal single-file habit tracker with Supabase cloud sync.

## Files

- `index.html` - the app file for GitHub Pages.
- `supabase_habit_sync_setup.sql` - database setup for Supabase sync.

## Daily Use

1. Open the GitHub Pages URL on PC or phone.
2. Open Settings in the app.
3. Enter your Supabase URL, public anon key, and private sync code.
4. Use `Load Cloud` to pull your latest data.
5. Enable `Auto Sync after changes` and `Load Cloud on app start`.

## Important

GitHub stores the app code. Supabase stores the app data.

Do not commit exported data files or private sync codes.
